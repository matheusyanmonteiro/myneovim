# Keymap reference

`<leader>` is the Space key.

## General and navigation

| Key | Action |
| --- | --- |
| `<leader>w` | Save file |
| `<leader>q` | Close window |
| `<C-h/j/k/l>` | Move between windows |
| `<leader>e` | Toggle Neo-tree |
| `<leader>ff` | Find files |
| `<leader>fg` | Search text in project |
| `<leader>fb` | Find open buffers |
| `[b` / `]b` | Previous / next buffer |
| `<leader>bd` | Delete current buffer safely |
| `<leader>tt` / `<C-\>` | Toggle terminal |
| `<Esc><Esc>` | Leave terminal mode |

## LSP

| Key | Action |
| --- | --- |
| `gd` | Definition |
| `gD` | Declaration |
| `gr` | References |
| `gI` | Implementation |
| `K` | Hover documentation |
| `<leader>rn` | Rename symbol |
| `<leader>ca` | Code action |
| `<leader>lf` | Format buffer |
| `<leader>ld` | Line diagnostic |
| `[d` / `]d` | Previous / next diagnostic |

## Git

| Key | Action |
| --- | --- |
| `[h` / `]h` | Previous / next hunk |
| `<leader>hs` | Stage hunk |
| `<leader>hr` | Reset hunk |
| `<leader>hp` | Preview hunk |
| `<leader>gb` | Blame current line |
| `<leader>gB` | Toggle continuous blame |
| `<leader>gd` | Open Diffview |
| `<leader>gH` | File history |
| `<leader>gc` | Close Diffview |

## C++ and debugging

| Key | Action |
| --- | --- |
| `<leader>cc` | Compile current C++ file |
| `<F5>` | Start / continue |
| `<F10>` | Step over |
| `<F11>` | Step into |
| `<F12>` | Step out |
| `<leader>db` | Toggle breakpoint |
| `<leader>dB` | Conditional breakpoint |
| `<leader>du` | Toggle DAP UI |
| `<leader>dr` | Open debug REPL |
| `<leader>dt` | Terminate debug session |

## Markdown, sessions and VICARIOUS

| Key | Action |
| --- | --- |
| `<leader>mr` | Toggle rendered Markdown |
| `<leader>mp` | Rendered Markdown side preview |
| `<leader>ss` | Restore project session |
| `<leader>sS` | Select session |
| `<leader>sl` | Restore last session |

| `<leader>ui` | Study Mode status |
